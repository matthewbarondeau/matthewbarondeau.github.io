Self Monitoring Stress System Fall 2019 - Spring 2020

  • This project was part of the senior capstone course for 2019-2020 at UT Austin. Under Dr. Emily Porter, our team designed and built an embedded system and mobile application. The embedded processor, a TI MSP432P401R, was placed on a PCB. This PCB was sewn into a glove that a user will wear. The system collects heart rate, movement, and GSR and communicates the data through bluetooth to a mobile application. The mobile application then uses a knn model to classify the user as stressed or unstressed and communicate the status to them.
  • More information about the embedded system can be found here: SMSS
  • More information about the mobile application and model can be found here: IOS App

Tron Handheld video game Spring 2017

  • This project was developed for EE319k for the TI TM4C123GH6PM. In this project, two players would compete in a head to head TRON light cycle game. The users would perform input using two ADCs sampling an analog joystick as well as two bottons. While the players competed, audio continuously played in response to in game events.
  • More Information can be found here: TronV2
