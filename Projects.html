Self Monitoring Stress System Fall 2019 - Spring 2020

  • Senior design project at UT Austin under Dr. Emily Porter

  • Sensors collected heart rate, movement, and GSR
  • Sensors, MSP432, and power systems were laid out on PCB
  • More information about the embedded system can be found here: SMSS
  • More information about the mobile application and model can be found here: IOS App
