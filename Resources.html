Projects

Machine Learning Hardware Accelerator Fall 2021

  • Implemented openMP and hardware accelerator optimizations of darknet on ultra96 board
  • Achieved 25x speedup over baseline implementation
  • Project can be found here: ML_Accelerator

A Survey of Multiplier Rounding Techniques Fall 2021

  • Surveyed multiplier rounding techniques for accuracy tradeoffs in consecutive multiplier algorithms
  • Project source code can be found here: Rounding_Repository
  • Paper can be found here: Rounding_Paper

Parallelization of the Barnes-Hut Algorithm Spring 2021

  • Developed CPU and GPU versions of Barnes-Hut algorithm
  • Experimented with openMP to accelerate CPU versions
  • Achieved 10x speedup over CPU version including tree construction and data transfer
  • More Information can be found here: Barnes-Hut

AES Hardware Accelerator Fall 2019

  • Developed AES accelerator for Zedboard as part of EE382N-4 at UT Austin
  • Wrote Verilog state machine to control AES core and Software to offload computation
  • More Information can be found here: AES

Real Time Operating System Spring 2019

  • Real-Time Operating System developed for TI TM4C123GH6PM
  • Originally design during EE445M course, but has been cleaned up and flushed out
  • Components include UART Interpreter, Priority Scheduler, Process Loader, and PI Control Application

Self Monitoring Stress System Fall 2019 - Spring 2020

  • Senior design project at UT Austin under Dr. Emily Porter
  • Designed system consisting of MSP432, iOS application, and network of sensors to measure and track stress
  • Sensors collected heart rate, movement, and GSR
  • Sensors, MSP432, and power systems were laid out on PCB
  • More information about the embedded system can be found here: SMSS
  • More information about the mobile application and model can be found here: IOS App

Tron video game Spring 2017

  • Originally designed for EE319k fial competition
  • Overhauled to include CAN based multiplayer, SD card storage, and cleaned up code
  • Two players play head to head over CAN in the classic game TRON
