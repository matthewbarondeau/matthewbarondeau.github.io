EE306 Tasks Fall 2018

  • Designed 5 tasks for Dr. Yerraballi's EE306 course to supplement homeworks
  • Task 1 involves implementing a seven segment display using logisim
  • Task 2 involves designing two state machines using JFLAP
  • Task 3 involves designing an ALU and Register File using logisim
  • Task 4 involves extending Task 3 to include an instruction decode and bus
  • Task 5 introduces the students to git and GitHub
  • The tasks can be found here: Tasks
