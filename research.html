Research

Research Interests

  • Developing low-cost context switching mechanisms to improve the utilization of lightweight accelerator cores
  • Improving the operating system to accelerator interface to allow for better system efficiency

Course Projects

Evaluating Runahead Execution with Data Prefetching Spring 2022

  • Evaluated precise runahead execution in concert with data prefetchers using SCARAB simulator
  • Demonstrated that runahead execution enables better timeliness for data prefetchers
  • Final report can be found here: Runahead Paper

Machine Learning Hardware Accelerator Fall 2021

  • Implemented 40x faster inference of darknet on ultra96 board using openMP and hardware accelerator
  • Project source code can be found here: ML Accelerator

A Survey of Multiplier Rounding Techniques Fall 2021

  • Surveyed multiplier rounding techniques for accuracy tradeoffs in consecutive multiplier algorithms
  • Project report can be found here: Rounding Paper

Parallelization of the Barnes-Hut Algorithm Spring 2021

  • Developed 10x faster implmententation using openMP CPU and GPU versions of Barnes-Hut algorithm
  • Project source code can be found here: Barnes-Hut

AES Hardware Accelerator Fall 2019

  • Developed AES accelerator for Zedboard as part of EE382N-4 at UT Austin
  • Project source can be found here: AES

Self Monitoring Stress System Fall 2019 - Spring 2020

  • Senior design project at UT Austin under Dr. Emily Porter
  • Designed system consisting of MSP432, iOS application, and network of sensors to measure and track stress
  • Embedded system source can be found here: SMSS
  • Mobile application can be found here: IOS App
